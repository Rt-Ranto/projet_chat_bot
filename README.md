# 🚀 ChatBot-GG

> **Statut :** En cours de développement (80%)

## 📝 Description
Application web de chat intelligent utilisant l'API Gemini(gemini-2.5-flash-lite) et Groq(llama-3.3-70b-versatile) 
pour fournir des réponses en temps réel, avec une interface moderne et responsive.*

## ✨ Fonctionnalités
- [x] Interface de chat interactive (React + Tailwind)
- [x] Gestion de l'historique des conversations avec MongoDB
- [x] Intégration de l'API Gemini et Groq (Backend Flask)
- [x] Système d'authentification des utilisateurs
- [ ] Recherch de chat (À venir)
- [ ] Generation d'image (À venir)

## 🛠️ Technologies utilisées
- **Frontend :** React, TypeScript, Tailwind CSS, FontAwesome
- **Backend :** Flask, Python-dotenv, Aragon2
- **Base de données :** MongoDB / PyMongo
- **IA :** Google Generative AI (Gemini & Groq)

## ⚙️ Installation
1. Cloner le projet : `git clone [URL]`
2. Installer les dépendances : `pip install -r requirements.txt` et `npm install`
3. Configurer le `.env` (voir `.env.example`)
4. Lancer le serveur : `python app.py` et `npm run dev`
